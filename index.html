Introducción

Este documento HTML es común para toda la clase. Tu misión es crear un tema visual únicamente con CSS. No puedes modificar este HTML (salvo cambiar el título de la página si lo deseas), pero puedes usar cualquier selector válido de CSS.

Enfócate en tipografía, colores, espaciados y layout responsivo básico. Añade micro-interacciones para hover/focus sin romper la accesibilidad.

  • Nocturno minimalista
  • Retro noventero
